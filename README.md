# Sistema-analise-notas-portugol
-  Projeto desenvolvido em Portugol no Visualg

## Funcionalidades:
-  Cadastro de 5 alunos (se modificado de forma simples, pode ser mais)
-  Cálculo da média da turma
-  Identificação da maior e menor nota
-  Contagem de alunos acima de média

## Conceitos aplicados
-  Vetores
-  Estruturas de repetição
-  Estruturas condicionais
-  Funções
-  Procedimento
